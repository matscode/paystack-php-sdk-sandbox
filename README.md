# Paystack SDK Sandbox
![Readme Banner](https://github.com/matscode/paystack-php-sdk-sandbox/raw/main/public/readme-banner.png "Readme Banner")
This project's intent is to be a deposit of example usage of the [Paystack PHP SDK](https://github.com/matscode/paystack-php-sdk) and maybe serve as a boilerplate for your next payment solution. But, hey, you would have to do some refactoring/restructuring.

## Laravel
This requires your prior understanding of the Laravel Framework.
<br>
If you need help navigating your way around Laravel framework. Head to the official [Laravel documentation](https://laravel.com/docs).
